Jeu du pendu en Python

Objectif : Créer un jeu interactif permettant à l’utilisateur de deviner un mot.

Description : Le programme choisit un mot aléatoire dans une liste. L’utilisateur propose des lettres et le programme affiche l’état du mot et le nombre d’erreurs restantes.
